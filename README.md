Simple web game built with plain TypeScript.

### Running project at **http://localhost:3000/**:

```bash
npm run dev
```

### Generating docs:

```bash
npx typedoc
```

or

```bash
npm run docs
```

### Generating sprites with helper script:

```bash
python -m sprites.split_sprites
```

While specifying the **sprite_id** in the file. For each id, a config can be added and. stored in **configs.py**
